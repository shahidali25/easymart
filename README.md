# 🛒 EasyMart - E-Commerce Website

EasyMart is a full-stack e-commerce web application developed to provide users with a simple and secure online shopping experience. Customers can browse products, add items to the cart, place orders, and make online payments. The system also provides an admin panel for managing products, categories, and customer orders.

---

## 📌 Features

### 👤 User
- User Registration & Login
- Browse Products
- Search Products
- Product Categories
- Add to Cart
- Wishlist
- Checkout
- Online Payment
- Order History
- User Profile Management

### 🔑 Admin
- Secure Admin Login
- Add, Update & Delete Products
- Manage Categories
- Manage Orders
- View Customers
- Dashboard with Statistics

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/EasyMart.git
```

2. Move the project to the XAMPP `htdocs` folder.

3. Start **Apache** and **MySQL** using XAMPP.

4. Create a database named:

```
easymart
```

5. Import the SQL file into MySQL.

6. Open your browser:

```
http://localhost/EasyMart
```
